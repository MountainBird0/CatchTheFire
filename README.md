# 🔥 Catch The Fire
불장난을 좋아하는 고양이들과<br>
그 고양이들을 막기 위한 동물 친구들의<br>
좌충우돌 멀티 배틀 땅따먹기 게임<br>
<br>

# 🕹 게임 조작 방법 <br>
**Mobile**<br>
- 이동 : 좌측 조이스틱<br>
- 대시 : 대시 버튼 클릭<br>
- 기본공격 : 공격 버튼 클릭<br>
- 스킬공격 : 스킬 버튼 클릭<br>
<br>

# 🚩 게임 목표 <br>
- 상대방보다 많은 땅을 점령하세요<br>
- 레드팀은 불을 지른 만큼 점령할 수 있고<br>
- 블루팀은 불이 없애서 점령을 저지해야 해요<br>
<br>
